# Christmas Miracle 2022

Repo for Onlines julekalender 2022. Applikasjonen er bygget med Create React App og go.

Example env file (Root for docker compose and /backend for local dev):

```env
POSTGRES_USER=admin
POSTGRES_PASSWORD=admin
POSTGRES_DB=cm
DB_DIALECT=postgres
DB_PORT=5432
DB_HOST=psql
```

## Teknologier

### Frontend

- Create React App
- TypeScript
- ChakraUI

### Backend

- Go
- Mux
- Postgres
